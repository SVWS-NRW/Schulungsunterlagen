# Fortbildung: Betreute Installation von Proxmox für Einsteiger 


Inhalte:
+ Gemeinsame Installation von Proxmox auf einem Debian 13 - Server (auch für Schulungsserver sinnvoll) 
+ Auf dem Proxmox kann dann unter anderem ein WeNoM-Server installiert werden


Zielgruppe:
+ Einsteiger ohne Vorkenntnisse in Virtualisierung

Voraussetzungen:
+ Ein Server mit bereits installiertem Debian 13

Hinweise:
+ Anmeldung per E-Mail erforderlich. Bitte sendet eine E-Mail mit dem Betreff "Anmeldung Proxmox Fortbildung" an anne.schueller@svws-nrw.de.
