# SVWS-Schulungsunterlagen

Auf dieser Seite finden Sie Schulungsmodule zur Durchführung von Fortbildungen im Umgang mit dem SVWS-Server.
Passend zu den Modulen stehen anonymisierte Übungsdatenbanken für den Einsatz in Schulungen zur Verfügung.

Als Fachberaterin oder Fachberater bzw. als Fortbildende können Sie das Material unter einer Creative-Commons-Lizenz kostenfrei nutzen.

Für die Fachberatung und Fachmoderation finden Sie hier außerdem Anleitungen zum technischen Umgang mit den Schulungsumgebungen.
Ergänzend stehen Ihnen die „Tipps der Woche“ sowie eine Übersicht über die Mikrofortbildungen für Fachberatungen zur Verfügung.
