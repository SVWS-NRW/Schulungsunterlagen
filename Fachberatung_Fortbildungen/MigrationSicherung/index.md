# Fortbildung: Migration und Datanbanksicherung



### Inhalte

+ Migrieren einer SchILD2-Datenbank mit Hilfe des Admin-Clients
+ Erstellen einer Verbindungsdatei (con-Datei) für SchILD-NRW 3
+ Datenbank-Sicherung durchführen und Einspielen einer Sicherungs-Kopie über den Admin-Client
+ Mögliche Probleme bei der Migration


### Ablauf der Fortbildung
Die oben genannten Inhalte werden zunächst vorgeführt. Auf mögliche Fehlerquellen wird hierbei hingewiesen. Dies dauert ca. 45 Minuten.

Im Anschluss können Fragen gestellt werden und/oder die gezeigten Inhalte selbst einmal "geübt" werden - sofern eine lokale Installation vorhanden ist. Wer nicht viel Zeit hat oder keine Fragen hat kann das Meeting einfach verlassen.
