# 9. Reiter: Reportverwaltung

Die Reportverwaltung ermöglicht den Druck von Formularen, die über die Standardformulare hinausgehen.
Im Fenster können links mit den Standardfiltern für den Container schnell einzelne Schüler:innen oder Gruppen ausgewählt werden.
Rechts ist die Ordnerstruktur des Ordners `.\SVWS-Arbeitsverzeichnis\Schild-Reports` dargestellt. In dieses Verzeichnis hinein können Sie neue Reports z. B. mit Hilfe des Windows-Explorers einfügen.
Reports für SchILD werden auf dieser Seite bereitgestellt:
[Schild-Reports](https://www.svws.nrw.de/svws-server-schild-nrw-3/reports).

## 9.1 Menüband

Im [Menüband](https://schulverwaltungsinfos.nrw.de/svws/wiki/index.php?title=Men%C3%BCband_(Reportverwaltung)) können einige wenige Einstellungen vorgenommen und Informationen abgerufen werden.

Ein Klick auf "Standardformulare" zeigt z. B., welche Berichte als Standard für Schulbescheinigungen oder Stammblätter eingestellt sind.

Ein Rechtsklick auf die Formulare ermöglicht einige Funktionen, die für dieses spezielle Formular durchgeführt werden können.

## 9.2 Aufgaben zum Reiter: Reportverwaltung

**Aufgabe 1**
Starten Sie den Report "Kursliste - Klasse Namen.rtm". Wählen Sie einen Kurs aus und dann "erst Vorschau", um sich den Bericht anzusehen.

**Aufgabe 2**
Sehen Sie sich den Bericht "Schülerliste - Klasse.rtm" für die Klasse 05A an.
Sehen Sie sich nun denselben Bericht *aus dem vergangenen Schuljahr* an.

**Aufgabe 3**
Aktivieren Sie den [Schnellzugriff](https://schulverwaltungsinfos.nrw.de/svws/wiki/index.php?title=Reports_verwenden_(Einf%C3%BChrung_in_SchILD-NRW)#Einstellungen_zu_Reports.2FSchnellzugriff) auf häufig genutzte Reports. Hiernach ist ein Neustart des Programms nötig.
Suchen Sie einen Report einer Klassenliste und fügen Sie ihn diesem Schnellzugriff hinzu.

**Aufgabe 4**
Wählen Sie einen Report einer Schulbescheinigung als "Standard-Bescheinigung".
Sehen Sie sich die Vorschau aus dem Reiter "Schüler" heraus an.

**Aufgabe 5**
Drucken Sie einen [Serienbrief](https://schulverwaltungsinfos.nrw.de/svws/wiki/index.php?title=Serienbriefsammlung_(Reportverwaltung)) an die Erziehungsberechtigten der Klasse 08A und teilen Sie ihnen mit, dass sich wegen der Schließung der Einrichtung durch das Gesundheitsamt die Klassenfahrt um vier Wochen verschiebt.
