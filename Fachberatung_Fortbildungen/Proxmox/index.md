# Fortbildung: Betreute Installation von Proxmox für Einsteiger 


Inhalte:
+ Gemeinsame Installation von Proxmox auf einem Debian-Server (auch für Schulungsserver sinnvoll) 
+ Auf dem Proxmox kann dann eine WeNoM-Server instaliert werden


Zielgruppe:
+ Einsteiger ohne Vorkenntnisse in Virtualisierung


Hinweise:
+ Wer nur "zuschauen" möchte, ist ebenfalls herzlich willkommen.
