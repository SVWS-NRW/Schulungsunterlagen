# Fortbildung: Betreute Installation von VirtualBox für Einsteiger 


Inhalte:
+ Gemeinsame Installation eines WeNoM-Servers auf einem privaten oder von der Bezirksregierung bereitgestellten **Windows-Rechner**.
+ Ziel ist, dass ihr nach der Fortbildung einen laufenden WeNoM-Server auf eurem Rechner habt, um damit zu experimentieren.


Zielgruppe:
+ Einsteiger ohne Vorkenntnisse in Virtualisierung

    
Hinweise:
+ Anmeldung per E-Mail erforderlich. Bitte sendet eine E-Mail mit dem Betreff "Anmeldung VirtualBox Fortbildung" an anne.schueller@svws-nrw.de.
